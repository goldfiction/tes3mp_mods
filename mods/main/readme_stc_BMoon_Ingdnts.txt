     1. INSTALLING THE PLUGIN

*****************************************************************

To install the plugin, unzip the files into the Morrowind/Data Files
directory. 


*****************************************************************

     2. PLAYING THE PLUGIN

*****************************************************************

BLOODMOON is required (Duh).

From the Morrowind Launcher, select Data Files and check the box 
next to the stc_Bloodmoon_Ingredients.esp file.

I was a bit annoyed to find out that none of the new ingredients in Bloodmoon 
could be bought from Fort Frostmoth, never mind in infinite amounts.

This adds a new NPC to the Imperial Cult Shrine that sells infinite amounts of most 
of the new ingredients. There is no Stahlrim or Snow Bear/SNow Wolf Belts. Sorry, that's
 still going to be kept rare. :)
