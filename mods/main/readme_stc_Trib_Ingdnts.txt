     1. INSTALLING THE PLUGIN

*****************************************************************

To install the plugin, unzip the files into the Morrowind/Data Files
directory. 


*****************************************************************

     2. PLAYING THE PLUGIN

*****************************************************************

TRIBUNAL is required (Duh).

From the Morrowind Launcher, select Data Files and check the box 
next to the stc_Tribunal_Ingredients.esp file.

I was a bit annoyed to find out that only three of the new ingredients in Tribunal 
could be bought in infinite amounts, the rest in a handful of plants scattered around.

This adds a chest to the Mournhold magic shop that has infinite amounts of most 
of the new ingredients. There is no Adamantium Ore. Sorry, that's still going to be
 kept rare. :)

The other three ingredients can be found in the Temple Infirmary.
